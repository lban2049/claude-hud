# User Intent

## Target Audience
Developers who use Claude Code daily and want better visibility into their session — context usage, tool activity, agent progress, and rate limits.

## Use Cases
- Installing and configuring Claude HUD for the first time
- Customizing the HUD display (presets, individual toggles, colors, layout)
- Understanding what each HUD element shows and how to interpret it
- Troubleshooting common issues (Linux install, usage not showing, config errors)
- Contributing to Claude HUD development

## Documentation Focus
This documentation takes the form of a **product guide**:
- Quick start installation and setup
- Visual walkthrough of HUD elements and what they mean
- Complete configuration reference with examples
- Troubleshooting guide for common issues
- Architecture overview for contributors
